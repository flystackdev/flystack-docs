---
title: "Flight Lookup"
api: "GET https://api.flystack.dev/v1/flights/lookup"
description: "Lookup a single flight and get detailed information by IATA or ICAO flight code."
---

The Flight Lookup API combines schedule data, near real-time tracking, and aircraft details for a **single** scheduled & live flight. Itâ€™s ideal for flight detail screens: status changes, times, gates, terminals, and aircraft information in one response.

<Note>
  This endpoint returns the closest matching flight (live, scheduled, or landed). If you need all occurrences of a flight number across a day, use the [Flight Schedules API](/api-reference/endpoints/schedules) and query by `flight_iata` or `flight_icao`.
</Note>

### Authentication

<ParamField header="x-api-key" type="string" required={true}>
  Your FlyStack API token. You can create one from the dashboard (see [Authentication](/authentication)).
</ParamField>

### Query parameters

<ParamField query="flight_icao" type="string" required={false}>
  Search by flight ICAO code-number (e.g., `AAL6`).
</ParamField>

<ParamField query="flight_iata" type="string" required={false}>
  Or search by flight IATA code-number (e.g., `AA6`).
</ParamField>

<Note>
  Provide either `flight_iata` or `flight_icao`.
</Note>
