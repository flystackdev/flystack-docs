---
title: "Real-Time Flights"
api: "GET https://api.flystack.dev/v1/flights"
description: "Track live flights worldwide with near real-time position, speed, heading, and status."
---

Integrate real-time flight tracking (live ADS-B style data) into your product: monitor individual flights, power flight status screens, visualize traffic on a map, or analyze airspace density and regularity.

### Authentication

<ParamField header="x-api-key" type="string" required={true}>
  Your FlyStack API token. You can create one from the dashboard (see [Authentication](/authentication)).
</ParamField>

### Query parameters

#### Area & performance

<ParamField query="bbox" type="string" required={false}>
  Bounding box: `south_lat,south_lng,north_lat,north_lng`.
</ParamField>

<ParamField query="zoom" type="number" required={false}>
  Map zoom level to reduce the number of returned flights (0â€“11).
</ParamField>

#### Aircraft / airline / flight filters

<ParamField query="hex" type="string" required={false}>
  Filter by ICAO24 hex address.
</ParamField>

<ParamField query="reg_number" type="string" required={false}>
  Filter by aircraft registration number.
</ParamField>

<ParamField query="airline_icao" type="string" required={false}>
  Filter by airline ICAO code.
</ParamField>

<ParamField query="airline_iata" type="string" required={false}>
  Filter by airline IATA code.
</ParamField>

<ParamField query="flag" type="string" required={false}>
  Filter by airline country ISO-2 code.
</ParamField>

<ParamField query="flight_icao" type="string" required={false}>
  Filter by flight ICAO code-number.
</ParamField>

<ParamField query="flight_iata" type="string" required={false}>
  Filter by flight IATA code-number.
</ParamField>

<ParamField query="flight_number" type="string" required={false}>
  Filter by flight number only.
</ParamField>

<ParamField query="dep_icao" type="string" required={false}>
  Filter by departure airport ICAO code.
</ParamField>

<ParamField query="dep_iata" type="string" required={false}>
  Filter by departure airport IATA code.
</ParamField>

<ParamField query="arr_icao" type="string" required={false}>
  Filter by arrival airport ICAO code.
</ParamField>

<ParamField query="arr_iata" type="string" required={false}>
  Filter by arrival airport IATA code.
</ParamField>

#### Response shaping

<ParamField query="_fields" type="string" required={false}>
  Comma-separated list of fields to return (e.g., `hex,airline_iata,lat,lng`).
</ParamField>

<ParamField query="_view" type="string" required={false} default="object">
  View format:
  - `object`: default JSON objects
  - `array`: array rows (useful for browsers / fast rendering)
</ParamField>

<Note>
  For `_view=array`, make sure `_fields` lists fields in the exact order you want in the returned arrays.
</Note>
